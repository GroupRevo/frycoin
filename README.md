What is Frycoin?
----------------

Frycoin is a lite version of Bitcoin using scrypt as a proof-of-work algorithm.
 - 2 minute block targets
 - 500 million total coins
 - 300 coins per block
 - Difficulty retargets every block - Dark Gravity Wave
